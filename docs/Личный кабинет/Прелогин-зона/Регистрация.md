# Регистрация

## Создание новой заявки

### `Шаг № 1` – Данные представителя
**На этапе сбора информации о представители мерчант указывает данные:**
- ФИО *(обязательно)*
- Номер телефона *(обязательно)*, *(требует подтверждения)*
- Почта

### `Шаг № 2` – Данные сети
**На этапе сбора информации о компании мерчант указывает данные:**
- ИНН *(обязательно)*
- Тип юр. лица *(обязательно)*
- Название *(обязательно)*

## `Шаг № 3` – Завершение сценария заявки

После ввода корректных данных, подтверждения номера телефона. Пользователь переходит на статусный экран **"Ожидайте ответа администратора"** с возможностью авторазции в личном кабинете.

[**Статус верефикации сети:**](/)
```json
status_verification: "invite"
```

[**Статус видимсоти сети:**](/)
```json
status_published: "hidden"
```

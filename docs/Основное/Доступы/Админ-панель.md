# Админ-панель

Вход в админ-панель доступен только сотрудникам appoint. Каждый сотрудник, зарегистрированный в системе, имеет:
- ID
- ФИО
- Должность
- Номер телефона
- Электронную почту

## Авторизация в админ-панели

Для входа в админ-панель необходимо:
1. Перейти по адресу: [admin.appoint.ru](https://admin.appoint.ru)
2. Ввести корпоративные логин и пароль
3. Выбрать способ двухфакторной аутентификации:
   - **Получение кода по SMS** на привязанный номер телефона
   - **Получение кода по email** на корпоративную электронную почту
4. Ввести полученный код подтверждения для завершения авторизации

:::warning Важно!
Код подтверждения действителен в течение 5 минут. После истечения этого времени необходимо запросить новый код.
:::

## Создание нового пользователя админ-панели

Создание нового пользователя происходит по заявке в команду девопсов. Для подачи заявки требуется заполнить форму и отправить ее в чат **[PROD] appoint**

### Форма заявки на регистрацию нового сотрудника Appoint:

**Обязательные поля:**
- ID сотрудника
- ФИО полностью
- Должность *(роль сотрудник)*
- Номер телефона (в формате +7-XXX-XXX-XX-XX)
- Электронная почта *(корпоративная, например: mail@appoint.ru)*

## Восстановление доступа

В случае утери доступа к админ-панели:
1. Обратиться к непосредственному руководителю
2. Направить заявку в чат **[PROD] appoint**
3. Указать причину восстановления доступа и контактные данные

## Безопасность

- Запрещено передавать учетные данные третьим лицам
- Рекомендуется регулярно менять пароль
- При подозрении на компрометацию учетной записи немедленно сообщить в службу безопасности
- Коды подтверждения являются конфиденциальной информацией и не подлежат разглашению

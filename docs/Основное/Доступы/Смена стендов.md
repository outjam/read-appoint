# Смена стендов

## Стенды разработки
В мобильном приложении **appoint** можно переключаться между стендами:

:::info Стенды разработки
- `DEV` — стенд разработки, использует базу данных `DEV`.

- `STAGE` — стенд предрелиза, использует базу данных `PROD`.

- `PROD` — стенд официального релиза приложения.
:::

## Смена стенда
:::danger Внимание!
Смена стенда доступна только для приложения, установленного из TestFlight
:::

**Чтобы сменить текущий стенд в мобильном приложении appoint:**
1. Перейти в профиль пользователя.
2. Тапнуть по имени пользователя.
3. На экране редактирования пользователя проскроллить страницу до конца *(дойдя до номера версии приложения, например: appoint 1.0.1)*.
4. Быстрый тап **более 10 раз** откроет `DEBUG SCREEN`, где доступен список стендов для переключения.

## Индикатор стенда
Во всём мобильном приложении присутствует индикатор текущего стенда. Он отображается под островом/чёлкой *(Dynamic Island)* и появляется, когда пользователь делает скриншот приложения, для быстрой идентификации стенда, на котором находится пользователь.

:::info Визуальный индикатор стенда
- `DEV` — в индикаторе выводится `develop`, красный цвет.

- `STAGE` — в индикаторе выводится `stage`, оранжевый цвет.

- `PROD` — в индикаторе выводится `appoint`, синий цвет.
:::

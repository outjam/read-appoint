# Белый список

В мобильном приложении **Appoint** реализована система **«Белый список»** для авторизации в приложении без подтверждения **СМС-кодом**.

:::danger Внимание!
Добавленные номера в **«Белый список»** будут работать только на `DEV` и `STAGE` стендах.
:::

## Добавление и управление белым списком

- Добавление и редактирование белого списка происходит в **[админ-панели](https://admin.appoint.ru)** администратора **Appoint** в разделе **«Белый список»**

- Администратор **Appoint** может добавлять новые номера телефонов для авторизации или удалять старые номера телефонов

- При добавлении номера телефона администратор указывает:
  - Номер телефона в формате `+7-000-000-00-00`
  - ФИО пользователя для авторизации без СМС-кода

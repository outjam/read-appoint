# Требования к файлу с платёжными реквизитами контрагента

## Назначение файла
Файл используется для автоматического добавления компании в систему как контрагента с возможностью:
- отправки платежей
- выставления счетов
- формирования бухгалтерских документов

---

**Разрешённые форматы:**
- **PDF** (`.pdf`)
- **CSV** (`.csv`, кодировка — UTF-8)
- **XLSX** (`.xlsx`)
- **JSON** (`.json`)
- **Microsoft Word** (`.doc`, `.docx`)

---

## Обязательные поля
Документ должен содержать следующие поля:

| Поле (системное имя)  | Название в документе     | Обязательность | Описание                                  |
| --------------------- | ------------------------ | -------------- | ----------------------------------------- |
| company_name          | Наименование организации | ✅              | Полное юридическое название               |
| inn                   | ИНН                      | ✅              | Идентификационный номер налогоплательщика |
| kpp                   | КПП                      | ✅              | Код причины постановки на учёт            |
| bank_name             | Банк                     | ✅              | Полное наименование банка                 |
| bic                   | БИК                      | ✅              | Банковский идентификационный код          |
| settlement_account    | Расчётный счёт           | ✅              | Р/с компании в банке                      |
| correspondent_account | Корреспондентский счёт   | ✅              | Корр. счёт банка                          |


## Необязательные поля
| Поле          | Название в документе | Описание                   |
| ------------- | -------------------- | -------------------------- |
| legal_address | Юридический адрес    | Адрес регистрации компании |
| bank_address  | Адрес банка          | Юридический адрес банка    |
| ogrn          | ОГРН / ОГРНИП        | Регистрационный номер      |

---
